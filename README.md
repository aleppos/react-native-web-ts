# React Native Web App TypeScript And GraphQL ready.

## Installation

```sh
$ yarn install


# Run Web/Ios/Android development
# Web
$ yarn start

# IOS (simulator)
$ yarn start ios

# Android 
$ yarn start android

```

## Additional includes

- [React Native Elements](https://react-native-elements.github.io/react-native-elements/)
- [Jest](https://jestjs.io/)
